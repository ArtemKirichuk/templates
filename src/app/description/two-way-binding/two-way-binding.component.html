Позволяет одновременно передавать и получать значения между родительским и дочерним компонентами.

Синтаксис

box in banana [()]

Принцип работы

Свойство @Output() использует в шаблоне inputChange, где input имя свойств @Input(). Например, если свойство @Input() имеет значение size, свойство @Output() должно иметь значение sizeChange.
