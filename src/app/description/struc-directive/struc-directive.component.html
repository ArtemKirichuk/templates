Построение атрибутной директивы

Эта секция про создание атрибутной директивы которая будет выделять название host элемента

  • Создание директивы cli ng generate directive 'name' (ng g d 'name'). В декораторе есть свойство указывающщее на CSS селектор директивы.
  • Импортируем ElementRef из core. ElementRef даёт доступ к DOM элементу директивы через свойство nativElement.
  • Добавляем ELementRef в конструтор чтобы ввести ссылку на host dom элемент. Элемент к которому вы применили селектор директивы.
  • Добавляем необходимую логику в конструкторе директивы.

Деактивация Angular обработки NgNonBindable

NgNonBindable - деактивирует антерполяцию, директивы и привязки в шаблонах. применяя NgNonBindable на элемент так же применяется на дочернии элементы.Однако NgNonBindable позволяет работать директивам.
