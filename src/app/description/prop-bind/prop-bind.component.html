Привязка свойств позволяет устанавливать сво-ва HTML элементов или директив. использую привязку свойств можно делать такие вещи как:

  • toggle функционал
  • задавать пути к ресурсам
  • и обменивать значениями между компонентами

Поток данных

Привязка свойств перемещает значение в одном направлении, из свойства компонента в свойства элемента.

Привязка свойств

Привязываемое свойство для задания значения заключается в квадратные скобки.Например можно привязать путь к картинке.

Символ квадратных скобок [] позволяет angular оценивать правую часть как динамическое выражение. Без скобок правая часть обрабатывается как строковый литерал и устанавливает свойство в статическое значение.
